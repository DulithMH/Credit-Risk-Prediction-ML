# Credit Risk Prediction Project

## Project Overview
This project predicts the likelihood of a loan applicant defaulting based on historical banking data. It follows a rigorous financial modeling pipeline to ensure statistical stability.

## Key Features
- **VIF Analysis:** Reduced multicollinearity from 200+ down to < 2.0.
- **Data Preprocessing:** Handled outliers via Winsorization and applied Log Transformations.
- **Handling Imbalance:** Used `class_weight='balanced'` to achieve a 79% Recall for defaults.

## Technologies Used
- Python (Google Colab)
- Scikit-learn, Pandas, Statsmodels, Seaborn
